# 📂 [chap 7] 자료구조 (Data Structures)

### 1. 튜플 (Tuple) 🔒
- 리스트와 유사하지만 **변경이 불가능**합니다.
- 딕셔너리의 **키(Key)**로 사용할 수 있다는 장점이 있습니다.
- **패킹/언패킹**: `a, b, c = (1, 2, 3)` 처럼 값을 묶거나 풀 수 있습니다.

### 2. 세트 (Set) 🍱
- 수학의 집합과 같으며, **중복을 허용하지 않습니다.**
- 순서가 없기 때문에 인덱스로 접근할 수 없습니다.

### 3. 딕셔너리 (Dictionary) 📖
- **키(Key)**와 **값(Value)**이 한 쌍으로 저장됩니다.
- `d.get(key, default)`: 키가 없어도 오류 대신 기본값을 반환하여 안전하게 값을 가져옵니다.

### 4. 문자열 메소드 텍스트 처리 🔤
- `strip()`: 공백이나 특정 문자 제거
- `split()`: 기준에 따라 문자열을 쪼개 리스트로 변환
- `join()`: 리스트 요소를 하나로 합쳐 문자열 생성

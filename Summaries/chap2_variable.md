# 📂 [chap 2] 변수와 객체 (Variables & Objects)

### 1. 변수의 본질 🧠
- **정의**: 컴퓨터의 메모리 공간에 이름을 붙이는 것.
- **동작 방식**: 데이터가 **객체(Object)** 형태로 메모리에 저장되고, 변수는 그 객체를 가리키는 **참조값**을 저장합니다.

### 2. 불변 객체 vs 가변 객체 ⚖️
| 구분 | 불변 객체 (Immutable) | 가변 객체 (Mutable) |
| :--- | :--- | :--- |
| **특징** | 생성 후 변경 불가능 | 생성 후 변경 가능 |
| **종류** | 정수, 실수, 문자열, 튜플 | 리스트, 세트, 딕셔너리 |
| **메모리 동작** | 값이 바뀌면 새로운 객체를 만들어 참조를 옮김 | 객체 내부의 값만 바꾸며 주소는 유지 |

> **예시**: `x = 10`에서 `x = 11`로 바꾸면, 기존 10을 수정하는 게 아니라 **새로운 11**을 만들어 x가 거기를 가리키게 합니다. 반면 리스트 `L = [1, 2]`를 `[1, 3]`으로 바꾸면 주소값은 변하지 않습니다.

### 3. 주요 개념
- **객체**: 변수와 함수를 하나로 묶은 것.
- **반올림**: `round(변수명, 자리수)`
